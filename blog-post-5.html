Algunos programas que implementan manejo de cadenas.

Autor: Manuel Bernabel (23-0285)

El manejo eficiente de cadenas es una habilidad fundamental en la programación, ya que las cadenas de texto son una parte omnipresente en el desarrollo de software. En este artículo, exploraremos cómo implementar el manejo de cadenas en tres lenguajes de programación ampliamente utilizados: Python, JavaScript y Java.

A través de estos ejemplos, no solo aprenderemos cómo realizar operaciones básicas en cadenas, sino que también entenderemos las diferencias en enfoques y sintaxis entre Python JavaScript y Java en el manejo de este tipo de datos.

Sin más preámbulos, adentrémonos en la fascinante exploración del manejo de cadenas en la programación.

Python

Este programa en Python implementa el manejo de cadenas al invertir una cadena dada utilizando la técnica de slicing con índices negativos.

					    # Ejemplo de inversión de una cadena en Python
def reverse_string(input_string):
    return input_string[::-1]

def main():
    original_string = "Hola, mundo!"
    reversed_string = reverse_string(original_string)
    print("Cadena original:", original_string)
    print("Cadena invertida:", reversed_string)

if __name__ == "__main__":
    main()

JavaScript

En este ejemplo en JavaScript, se implementan dos funciones para manejar cadenas. La primera función verifica si una subcadena está presente en la cadena principal, y la segunda función extrae una subcadena a partir de índices especificados.

					    // Ejemplo de búsqueda de una subcadena en JavaScript
function findSubstring(mainString, subString) {
    return mainString.includes(subString);
}

// Ejemplo de extracción de una subcadena en JavaScript
function extractSubstring(mainString, startIndex, endIndex) {
    return mainString.substring(startIndex, endIndex);
}

// Uso de las funciones
let originalString = "Hello, world!";
let subString = "world";
console.log("La subcadena se encuentra:", findSubstring(originalString, subString));
console.log("Subcadena extraída:", extractSubstring(originalString, 7, 12));

Java

En este ejemplo en Java, se implementa el manejo de cadenas al invertir una cadena dada utilizando un bucle for y un StringBuilder para construir la cadena invertida

					    // Ejemplo de manipulación de cadenas en Java
public class StringManipulation {
    public static void main(String[] args) {
        String originalString = "¡Hola, mundo!";
        String reversedString = reverseString(originalString);
        System.out.println("Cadena original: " + originalString);
        System.out.println("Cadena invertida: " + reversedString);
    }

    // Método para invertir una cadena en Java
    public static String reverseString(String inputString) {
        StringBuilder reversed = new StringBuilder();
        for (int i = inputString.length() - 1; i >= 0; i--) {
            reversed.append(inputString.charAt(i));
        }
        return reversed.toString();
    }
}

Estos ejemplos ilustran cómo se pueden manipular y trabajar con cadenas en Python, JavaScript y Java, respectivamente. Cada lenguaje ofrece sus propias herramientas y enfoques para el manejo de cadenas, lo que permite a los desarrolladores realizar operaciones eficientes en texto según las necesidades de su aplicación.

Sin duda, el dominio del manejo de cadenas en diferentes lenguajes es una habilidad valiosa para cualquier programador.
